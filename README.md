## Урок 1. Модальное окно
Разберем какие есть варианты подключения JavaScript. 
Напишем функции для открытия и закрытия модального окна. Реализуем Модальное окно поиска и Прелоадера.

### ОБЯЗАТЕЛЬНОЕ ЗАДАНИЕ
	- Скачать исходники по ссылке под видео
	- Повторить код за уроком
	- Результат необходимо залить на github

	- При вводе данных в поле ввода выводить значение поля в консоль
	- Результат необходимо залить на github
